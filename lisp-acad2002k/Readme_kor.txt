*프로그램 설명 
 ~~~~~~~~~~~~~
iDwgTab™ 프로그램은 BizIcom에서 제작된 유틸리티로 사용상의 제약이 전혀없는
Free 프로그램으로 계속적인 버그수정과 기능 개선을 진행하고 있습니다.

iDwgTab™프로그램은 AutoCAD R2000 이상에서 동작하는 유틸리티 프로그램 입니다.
AutoCAD2000기반의 프로그램에서 멀티 윈도우를 쉽게 조작하도록 도와 주는 
프로그램입니다. 사용법이 간단하므로 별도의 연습이 필요없이 바로 사용할 수 
있습니다. 가급적 아래의 문구를 끝까지 읽어 보시기 바라겠습니다.

iDwgTab2000.arx는 AutoCAD (LT)2000/2000i/2002에서 동작하는 버젼입니다.
iDwgTab2004.arx는 AutoCAD (LT)2004/2005/2006에서 동작하는 버젼입니다.
iDwgTab2007.arx는 AutoCAD (LT)2007에서 동작하는 버젼입니다.

프로그램기능

- 컨텍스트 메뉴의 쉬운 수정기능 (iDwgTab.ini : 참조)
- 간편한 언어변환 기능 (iDwgTab.ini : 참조)
- 컨텍스트 메뉴에서 도면 썸네일 이미지 지원
- 탭의 툴팁에서 도면정보를 쉽게 체크가능
- 도면의 Layout을 바로 활성시키는 기능
- 열려진 도면중 활성된 도면만 열고 나머지 도면 닫기
- 열려진 모든 도면의 저장 (저장시 Zoom & Extents 옵션선택) 기능
- 다른폴더로 열려진 모든도면저장 기능
- 열려진 도면을 저장후 모든 도면닫기 기능
- 열려진 도면을 저장하지 않고 모든 도면닫기 기능
- 열려진 도면의 계단식 배열기능
- 수평 타일식
- 수직 타일식
- 아이콘 정렬
- 열려진 도면에 스크립트(SCR) 파일실행 기능
- 열려진 도면을 오름 정렬 기능
- 열려진 도면을 내림 정렬 기능
- 열려진 도면중에서 유사이름으로 도면찾기 기능
- 환경설정 대화상자 지원
- 사용자가 컨텍스트 메뉴를 직접 디자인 가능한 기능 (사용자 등록명령 간능)
- 선택된 도면의 상위디렉토리 열기 기능
- Ctrl, Shift키와 방향키를 이용한 탭의 순서 배치기능
- 탭컨트롤로 파일을 드레깅하면 열기를 실행
- 탭도킹 위치가 'bottom' 일때는 오른쪽 마우스의 메뉴가 역순으로 배치지원 (id : pillager)
- 도면을 열기할때 탭의 위치를 Sort하여 결정 (id : daum2704442)
- 이전에 열린도면의 쉽게 열기하는 기능 (id : kaka568)
- 사용을 원하지 않는 메뉴항목 숨김 기능.
- 엑셀프로그램 처럼 탭의 다중선택이 가능하도록 기능추가 (id : lechen, cloudx)
- 탭에서 닫기버튼 지원
- 탭아이콘에 DWG버전표시
- DWG버전 아이콘표시
- Ctrl + Tab키를 이용한 바로선택 기능지원
- 선택된 탭들에 대해서 스크립트 실행
- 폰트 크기를 이용해서 탭사이즈 수정가능

===================
* 다국어 지정방법 *
~~~~~~~~~~~~~~~~~~~

다국어를 쉽게 변경할 수 가 있습니다. iDwgTab.ini파일중에 #LANGUAGE 값을 지정 
하기만 하면 됩니다. 만약 영어로 지정하고자 원한다면 아래와 같이 ENGLISH를 
설정합니다. #LANGUAGE = ENGLISH, 
한글로 지정하고자 원한다면 #LANGUAGE = KOREAN 으로 합니다.
여기서 ENGLISH, KOREAN은 대상언어의 데이블 이름입니다.

================
*프로그램의 상세 기능
 ~~~~~~~~~~~~~~~

- iDwgTab™ 프로그램은 일반오토캐드및 LT에서도 작동됩니다. 
  (알림 : LT에서 ARX를 구동 시켜주는 유틸리티가 필요합니다.)

- DWG/DXF파일에 따른 구별된 아이콘 지원합니다.

- 탭키에서 오른쪽 마우스 버튼을 지원합니다.
  (close, close all, 윈도우 명령, ...)

- 활성 탭 키의 툴팁으로 전체 경로및 도면정보를 보여줍니다.

- 스크립트 파일을 사용할 수 있으므로 현재 열려진 모든 문서에
  일괄적으로 명령을 수행할 수 있습니다.

  예) 탭으로 마우스이동 -> 오른쪽 마우스 클릭 -> Script...를 선택
      스크립트 지정 -> 모든도면에 일괄 처리

      test.scr 
      ------- 
      zoom
      e
      plot

  (주의) 여러도면에 스크립트를 적용하시고자 할때 우선 대상 스크립트 루틴이
  올바르게 동작하는지를 체크하셔야 합니다. 만약 오류나 그밖의 문제로 인하여
  더이상 진행이 이루어지지 않게되면, 다른 도면에서도 동일한 문제점이 계속해서
  발생됩니다. 이런 상황에서는 모든 도면을 닫게되면 증상이 사라집니다.
   
- 도면 미리보기/파일정보 추가

- 열려진 도면의 정렬 기능

- 열려진 도면을 바로 찾아서 활성시켜주는 기능 추가. (예: "a*.dwg")
